# World Chain {{ className: 'text-5xl' }}

![World Chain](/images/docs/world-chain/worldchain-cover.png)

World Chain is a blockchain for humans. World chain offers several unique primitives: 

- Free gas fees for all verified humans 
- Native mobile distribution to all World App users through [mini-apps](https://minikit-docs.vercel.app/mini-apps)
- Simplified crypto transaction [through mini apps](https://minikit-docs.vercel.app/mini-apps/commands/verify) 
- Sybil resistance for developers via [World ID](https://minikit-docs.vercel.app/world-id)
- Airdrop of WLD tokens to all verified humans

These primates enable World Chain builders to build never before possible applications and reach a global audience. 

World Chain is part of the Optimism Superchain.
